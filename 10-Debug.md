# Debug

* Toggle breakpoint, F9
* Start/continue, F5
* Stop, Shift+ F5
* Stop in/out, F11 or Shift+ F11
* Step over, F10
* Show hover, Ctrl+ K Ctrl+ I (!clear)

# C# Debugging Setup

1. Install VS Code.  
1. Install .NET CLI 
1. Install C# Extension for VS Code
1. Get a .NET Core project
1. Add .vscode files
    * tasks.json
    * launch.json
